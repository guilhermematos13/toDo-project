# Todo! Project

This is an individual project in order to improve my learning

## ⚙️ Installation

Install Dependencies

```bash
  npm install
```

Run Project

```bash
  npm run dev
```

Generate build project

```bash
  npm build
```

## 🖥️ Technologies Used

- React (18.2.0)
- Typescript (4.9.3)

## 📚 Libraries Used

- Radix
- Axios
- React Hot Toast
- Phosphor React
- Tailwind

## 🚀 Features

- Responsiveness
- Change of State
- Use Tailwind for styles
- Connection with API
